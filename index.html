Selected work

⭐ Featured — Master's Thesis · 2024–2025

Data Drift in ML-Based Intrusion Detection Systems

Investigated feature, label, and concept drift across three major IoT cybersecurity benchmark datasets — UNSW-NB15, BoT-IoT, and TON-IoT. Built a comprehensive drift detection pipeline integrating statistical hypothesis testing (KS test, chi-squared), KL divergence, and embedding-based visualisation via PCA and UMAP. Demonstrated that static ML models degrade significantly under temporal data shifts, and quantified that degradation across evolving attack categories such as exploits and fuzzers.

Python scikit-learn PCA / UMAP Statistical testing UNSW-NB15 BoT-IoT TON-IoT
