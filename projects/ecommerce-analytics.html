💡 Key Business Insights

💰 Revenue Concentration

20% of customers drive 80% of revenue. Champions have 8x higher CLV than average.

📍 Geographic Gap

Central region has 3x growth opportunity (5% → 15% conversion potential).

📅 Seasonal Peak

40% of annual revenue in Q4. Critical planning window: August-September.

👥 Retention Crisis

45% of customers inactive. Month 3-6 is critical churn point.

📦 Category Margin

Electronics & Fashion: 15-16% margins vs 10% for Books & Toys.

💼 Strategic Oppty

Bundling high + low margin products increases AOV by 22%.

🚀 How to Run Locally

git clone https://github.com/danyalrahujo/ecommerce-sales-analytics.git
cd ecommerce-sales-analytics

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Generate data
python src/data_generator.py
python src/data_processor.py

# Launch dashboard
streamlit run visualizations/dashboards/main_dashboard.py

# Visit: http://localhost:8501
